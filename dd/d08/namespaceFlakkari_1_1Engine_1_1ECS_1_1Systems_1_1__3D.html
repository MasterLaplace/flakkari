void Flakkari::Engine::ECS::Systems::_3D::handle_collisions ( Registry &  r,
std::unordered_map< Entity, bool > &  entities 
)

Handles collisions between entities.

Parameters
rThe registry containing the entities to update.
entitiesThe updated entities map.

This function will handle collisions between entities. It will check if the entities are out of the skybox and resolve the collision. It will also check for collisions between entities with a BoxCollider component. If a collision is detected, it will resolve the collision.

This function will also handle the death of entities. If an entity's health reaches 0, it will be killed.

The entities map will store the updated or killed entity

  • false means the entity is killed
  • true means the entity was updated
