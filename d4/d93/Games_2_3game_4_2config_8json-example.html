Games/<game>/config.json

ResourceManager classThis class is used to load and store resources (teemplates) It is a singleton, so it can be accessed from anywhere in the code It is initialized with a config file path, which contains the paths to the resources The config file must be a JSON file, with the following structure:

ResourceManager::adddScene("Games/<game>/config.json", "<scene>");
auto template = ResourceManager::getTemplateById("<game>", "<scene>", "<templateId>");
